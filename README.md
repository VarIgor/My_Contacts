# MyContacts

MyContacts — это простое Android-приложение для управления списком контактов с поддержкой базы 
данных **Room**, а также возможностью добавления, редактирования, удаления и перетаскивания 
контактов (drag & drop) с помощью `RecyclerView` и `ItemTouchHelper`.

## Функционал

- Добавление нового контакта
- Редактирование существующего контакта
- Удаление контакта (свайп влево)
- Перетаскивание контакта для изменения порядка (drag & drop)
- Локальное хранение данных через **Room Database**
- Автоматическое обновление списка после изменений

## Технологии

- **Kotlin**
- **Android Jetpack**:
    - Room
    - ViewBinding
    - DataBinding
- **RecyclerView** + **ItemTouchHelper**
- **Multithreading**: `ExecutorService` + `Handler` (основной поток)
